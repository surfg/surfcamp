# SurfCamp — короткая админ-инструкция

## Редактирование существующего кемпа (не создавайте дубль!)

1. Зайти в админку: `http://146.190.31.111:8080/admin/`.
2. Слева в меню — раздел **«Серф-кемпы»** (`Camps → Серф-кемпы`).
3. В списке кликнуть по **названию** кемпа — откроется форма редактирования.
4. Внести правки → **«Сохранить»** (редирект к списку) или **«Сохранить и продолжить редактирование»**.

## Как убедиться, что кемп ещё не заведён

Перед созданием нового кемпа:

1. Открыть список серф-кемпов.
2. Поиск вверху работает по: **названию**, **slug**, **адресу**, **региону**, **стране**.
3. Если ничего не нашлось — можно создавать новый.

## Загрузка фото

- Фото привязываются к кемпу через секцию **«Изображения кемпов»** внизу формы.
- Первое фото с галочкой `is_main` становится обложкой.
- После сохранения оптимизированные WebP-копии создаются автоматически. Если фото не появилось на фронте —
  выполните на сервере: `docker compose -f docker-compose.prod.yml exec backend python optimize_images.py`.

## Широта / долгота

Необязательные поля. Если не заполнены — кемп просто не показывается на карте,
всё остальное работает штатно.

## B&B (только проживание)

Чтобы кемп появился в фильтре «Только проживание»:
1. Отметить `has_bed_breakfast`.
2. Указать `bed_breakfast_price` (цена за ночь в USD без уроков).
